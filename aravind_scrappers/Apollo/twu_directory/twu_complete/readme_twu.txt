Pre-Requisites
--------------

1. Selenium must be installed. (pip install selenium)

2. All the files (twu.py, myconfig.py) must be available in same folder.

3. Before executing the script, All valid values should be available in myconfig.py file.

Steps to execute
------------------

1. Make changes in myconfig.py file if required.

2. Open cmd and navigate to the folder where all the files are available.

3. Execute file by writing command : twu.py
